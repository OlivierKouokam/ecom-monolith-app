# ecom-monolith-app

✅ **Version et environnement**

Spring Boot version : 3.5.7
Java version : 21

✅ **Particularités du projet**

Utilise Spring Data JPA → accès base de données simplifié
Utilise Spring Data REST → API REST générée automatiquement
Base H2 → environnement léger (in-memory)
Lombok → réduit le code boilerplate
Actuator → pour monitoring (utile en production)

✅ **Configuration spéciale**

Lombok est configuré comme annotation processor
Lombok est exclu dans le packaging final (bonne pratique ✅)
